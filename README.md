Social Media Graph Database Project

A full-stack social media graph database system built using PHP, MySQL, Python, and Cytoscape.
This project demonstrates how relational database systems and graph visualization techniques can be combined to model and analyze social network interactions.

Features
User accounts and profile data
Posts, likes, and comments
Follow relationships between users
Clubs and club memberships
Hashtags connected to posts
SQL analytical queries
Graph visualization using Cytoscape
PHP frontend website
GraphML export support
